# Observer Pattern - Speed Sensor

This project is a simple C++ implementation of the Observer Design Pattern.

The `SpeedSensor` class acts as the publisher and notifies registered display modules whenever the vehicle speed changes.

## Features

- Observer Design Pattern
- Runtime Polymorphism
- Speed Display
- Battery Consumption Simulation
- Object-Oriented Programming

## Classes

- IDisplay
- SpeedSensor
- SpeedDisplay
- BatteryManager

## Language

- C++

## Purpose

This project was developed to practice object-oriented programming concepts such as interfaces, inheritance, polymorphism, and the Observer Design Pattern.
